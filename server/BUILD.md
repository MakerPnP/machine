# Building

## Errors and solutions

### Windows: `could not execute 'llvm-config' one or more times.

Caused my missing LLVM installation.

Known working version is 22.1.0

Install from https://releases.llvm.org/ - e.g. `LLVM-22.1.0-win64.exe `

* Add it to the system path for all users when asked.

Example binaries path: `C:\Program Files\LLVM\bin`
